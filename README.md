# Procesamiento ENUT 2024 – Uso del tiempo, género y empleo público

Este repositorio contiene el script reproducible utilizado para el procesamiento de la **Encuesta Nacional de Uso del Tiempo (ENUT 2024)** del INEI.

El análisis forma parte del estudio:
*La Mujer en el Servicio Civil Peruano, 2026*.

Sub capítulo: **“Tiempo, género y trabajo en el Servicio Civil Peruano”**  

## Contenido

- `script_enut_2024.Rmd` → script reproducible
- `procesamiento_enut_2024.html` → versión html ejecutada del análisis

## Procesamiento realizado

- Descarga de módulos ENUT
- Limpieza de bases de datos
- Construcción de identificadores de hogar y persona
- Construcción de bases maestras
- Cálculo de indicadores de uso del tiempo
- Generación de gráficos

## Fuente de datos
INEI – Encuesta Nacional de Uso del Tiempo 2024.
